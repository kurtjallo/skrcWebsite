# St Katharine Rural Connect

The official website for **St Katharine Rural Connect (SKRC)** — a community-rooted initiative strengthening social, emotional, and practical support systems for rural people and places.

## About

SKRC serves farmers, offshore workers, rural families, and community members through four pillars of support:

- **Community Wellness & Mental Health** — Individual support, peer groups, and resilience workshops
- **Agricultural Support & Resources** — Farm wellbeing, grants, and sustainability sessions
- **Offshore Worker Engagement** — Remote support, peer connection, and family resources
- **Community Connection & Development** — Events, volunteering, and local partnerships

## Tech Stack

- [Next.js 16](https://nextjs.org/) with React 19 and App Router
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Motion](https://motion.dev/) (Framer Motion) for animations
- [shadcn/ui](https://ui.shadcn.com/) component primitives
- [Zod](https://zod.dev/) for form validation
- TypeScript throughout

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
