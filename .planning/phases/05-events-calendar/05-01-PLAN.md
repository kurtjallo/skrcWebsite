**Event types (src/types/event.ts):**

Create and export the following types:

```ts
export type EventAudience = 'farmers' | 'offshore' | 'families' | 'community';
export type EventType = 'workshop' | 'social' | 'volunteer' | 'support';
export type EventFormat = 'in-person' | 'virtual' | 'hybrid';

export interface Event {
  slug: string;
  title: string;
  description: string;      // Full description for detail page (2-4 sentences)
  date: string;              // ISO date string e.g. "2026-04-15"
  time: string;              // Display time e.g. "10:00 AM"
  endTime?: string;          // Optional e.g. "12:00 PM"
  location: string;          // Venue or "Online" or "Online + [venue]"
  audience: EventAudience[]; // Can serve multiple audiences
  type: EventType;
  format: EventFormat;
  image?: string;            // Optional image path
  featured?: boolean;        // For homepage preview
}
```

Also export display-label maps for UI rendering:

```ts
export const AUDIENCE_LABELS: Record<EventAudience, string> = {
  farmers: 'Farmers',
  offshore: 'Offshore Workers',
  families: 'Families',
  community: 'All Community',
};

export const TYPE_LABELS: Record<EventType, string> = {
  workshop: 'Workshop',
  social: 'Social',
  volunteer: 'Volunteer',
  support: 'Support',
};

export const FORMAT_LABELS: Record<EventFormat, string> = {
  'in-person': 'In Person',
  virtual: 'Virtual',
  hybrid: 'Hybrid',
};
```

  <name>Task 2: EventCard component with premium hover effects</name>
  <files>src/components/events/EventCard.tsx</files>
  <action>
Create `src/components/events/EventCard.tsx` as a client component (`"use client"` for hover animations).

**Props:**
```ts
import { Event, AUDIENCE_LABELS, TYPE_LABELS, FORMAT_LABELS } from '@/types/event';

interface EventCardProps {
  event: Event;
}
```

**Structure:**
Render as `<article>` element wrapping a `<Link href={/events/${event.slug}}>`.

**Date badge (left or top-left):**
- Extract month abbreviation and day number from `event.date` using `new Date(event.date)`
- Display stacked: month (uppercase, text-xs, font-body, font-bold, text-accent-600) over day (text-2xl, font-heading, font-semibold, text-primary-900)
- Wrap in a `<time datetime={event.date}>` element for accessibility
- Badge area: `w-16 flex-shrink-0 flex flex-col items-center justify-center bg-stone-100 rounded-lg p-2`

**Card body (right of date badge or below on small screens):**
- Title: `<h3>` with `font-heading text-xl font-semibold text-primary-900 group-hover:text-primary-700 transition-colors`
- Time and location row: `<div>` with lucide `Clock` icon (size 14) + `event.time` (and endTime if present, formatted as "10:00 AM - 12:30 PM"), then lucide `MapPin` icon (size 14) + `event.location`. Use `text-text-muted text-sm font-body`.
- Tags row:
  - Event type tag: Small pill badge. Background color based on type:
    - workshop: `bg-accent-100 text-accent-600`
    - social: `bg-secondary-100 text-secondary-700`
    - volunteer: `bg-emerald-50 text-emerald-700` (use Tailwind default emerald)
    - support: `bg-blue-50 text-blue-700` (use Tailwind default blue)
  - Format indicator: Small pill with lucide icon -- `Monitor` for virtual, `MapPin` for in-person, `Wifi` for hybrid. `bg-stone-100 text-text-muted`
  - Audience tags: Render each audience as a small `text-xs` pill, `bg-stone-200 text-text-muted`
  - All pills: `rounded-full px-2.5 py-0.5 text-xs font-medium font-body`

**Card container:**
```
group relative bg-white rounded-xl border border-stone-200 p-5
flex flex-row gap-4
overflow-hidden
transition-all duration-300 ease-premium
hover:shadow-lg hover:-translate-y-1
```

**Gold top-border hover effect (Phase 2 signature):**
Add a `::before` pseudo-element via Tailwind:
```
before:content-[''] before:absolute before:top-0 before:left-0 before:w-full before:h-[3px]
before:bg-accent-500 before:scale-x-0 before:origin-left
before:transition-transform before:duration-400 before:ease-premium
group-hover:before:scale-x-100
```

**Reduced motion:** Wrap the hover translate in `motion-safe:hover:-translate-y-1`. The gold border and shadow are fine since they are visual-only, not motion-based.

**Responsive:** On mobile (`sm:` and below), stack vertically -- date badge on top left, card body below. Use `flex-col sm:flex-row`.

Import lucide icons: `Clock`, `MapPin`, `Monitor`, `Wifi`
Import `Link` from `next/link`
