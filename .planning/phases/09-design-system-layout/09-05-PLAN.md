    Create a large image card component with a curved white shape at the top (clip-path) and overlaid content at the bottom.

    **Props interface:**
    ```ts
    interface PreFooterCTAProps {
      heading: string;          // "Ready to *Transform* Your Community?" (supports EmphasisHeading syntax)
      body: string;             // Description paragraph
      ctaLabel: string;         // "Get Started Today"
      ctaHref: string;          // "/contact"
      backgroundImage: string;  // image path for next/image
      className?: string;
    }
    ```

    **Rendering:**
    ```tsx
    import Image from 'next/image';
    import { cn } from '@/lib/utils';
    import { EmphasisHeading } from './EmphasisHeading';
    import { CircleArrowCTA } from './CircleArrowCTA';

    export function PreFooterCTA({
      heading,
      body,
      ctaLabel,
      ctaHref,
      backgroundImage,
      className,
    }: PreFooterCTAProps) {
      return (
        <section className={cn('px-4 pb-8', className)}>
          <div className="relative overflow-hidden rounded-[20px]" style={{ minHeight: '500px' }}>
            {/* Background image */}
            <Image
              src={backgroundImage}
              alt=""
              fill
              className="object-cover"
            />

            {/* Dark gradient overlay from bottom */}
            <div className="absolute inset-0 bg-gradient-to-t from-primary-950/80 via-primary-950/40 to-transparent" />

            {/* Curved white shape at top -- creates the arch/wave transition from page bg */}
            <div
              className="absolute top-0 left-0 right-0 h-[80px] bg-surface-page"
              style={{
                clipPath: 'ellipse(55% 100% at 50% 0%)',
              }}
              aria-hidden="true"
            />

            {/* Content positioned at bottom-left */}
            <div className="relative z-10 flex min-h-[500px] flex-col justify-end p-8 md:p-12 lg:p-16">
              <EmphasisHeading
                text={heading}
                as="h2"
                className="text-3xl md:text-4xl lg:text-5xl text-white max-w-xl mb-4"
              />
              <p className="text-white/80 text-base md:text-lg max-w-lg mb-8">
                {body}
              </p>
              <CircleArrowCTA
                label={ctaLabel}
                href={ctaHref}
                variant="light"
              />
            </div>
          </div>
        </section>
      );
    }
    ```

    **Clip-path details:**
    - Use `clip-path: ellipse(55% 100% at 50% 0%)` on the curved white shape
    - This creates a wide ellipse anchored at the top center, giving a gentle arch that covers ~80px height at the peak
    - The `bg-surface-page` color matches the page background, so it visually "curves into" the image
    - The height of 80px gives enough curve without being too dramatic
    - `aria-hidden="true"` on the curve shape (purely decorative)
    - Do NOT use `clip-path: path()` or `shape()` -- `ellipse()` has the broadest browser support (97%+)

    **Layout details:**
    - Inset from page edges: `px-4 pb-8` (similar to InsetHero but with bottom padding to separate from footer)
    - Rounded corners: `rounded-[20px]` matching the hero pattern
    - Min height: 500px to ensure the image card is tall enough for the overlay content
    - Content at bottom: `flex-col justify-end` with generous padding
    - Heading max-width: ~xl (prevents too-wide headings)
    - Body text max-width: ~lg (comfortable reading width)

    **Important:**
    - SERVER component (no "use client")
    - Image alt="" because decorative (text overlay provides content)
    - The Image does NOT get `priority` -- this is below the fold
    - Uses EmphasisHeading and CircleArrowCTA from Plan 02
    - The `variant="light"` on CircleArrowCTA gives white text + white circle (for dark image overlay)
