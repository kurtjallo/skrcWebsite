**SkipLink (src/components/layout/SkipLink.tsx):**
- Renders an `<a href="#main-content">` with text "Skip to main content"
- Visually hidden by default using `sr-only` Tailwind class
- On focus: becomes visible with `focus:not-sr-only focus:absolute focus:top-2 focus:left-2 focus:z-[100] focus:px-4 focus:py-2 focus:bg-accent-500 focus:text-primary-950 focus:font-body focus:font-medium focus:rounded focus:outline-none focus:ring-2 focus:ring-primary-900`
- MUST be the first focusable element when rendered in layout

**UtilityBar (src/components/layout/UtilityBar.tsx):**
- Full-width bar at the very top of the page
- Background: `bg-primary-950` (deepest slate), text: `text-stone-100`
- Height: compact, `py-2`
- Layout: flex row, justify-between, max-w-7xl centered
- Left side: Phone icon (lucide Phone, size 14) + phone number as `<a href={SITE_CONFIG.phoneHref}>` with `tel:` link. Use `SITE_CONFIG` from constants.
- Right side: "Need Support?" link styled as `<a href="/contact">` with subtle underline on hover. Also a "Donate" link (lower priority, text-stone-400 color).
- Font: font-body, text-sm (14px)
- Phone link hover: text-accent-400 transition
- Import SITE_CONFIG from `@/lib/constants`

**Header (src/components/layout/Header.tsx):**
- Server component where possible, but gold underline hover is CSS-only so no client directive needed
- `<header>` element with `<nav aria-label="Main navigation">`
- Background: `bg-stone-50`, sticky top with `sticky top-0 z-50`, add subtle bottom border `border-b border-stone-200`
- Layout: max-w-7xl centered, flex row, items-center, justify-between, h-20
- Left: Logo text "St Katharine Rural Connect" -- use `font-heading font-semibold text-2xl text-primary-900`. Link to "/". No image logo yet, just text.
- Right: Desktop nav links from `NAV_ITEMS` (hide on mobile with `hidden lg:flex`). Each link:
  - `font-body font-medium text-base text-text-body`
  - Relative positioned, with `::after` pseudo-element for gold underline:
    ```css
    after:content-[''] after:absolute after:bottom-0 after:left-0 after:w-full after:h-[2px]
    after:bg-accent-500 after:scale-x-0 after:origin-left
    after:transition-transform after:duration-400 after:ease-premium
    hover:after:scale-x-100
    ```
  - Active link (matching current pathname): `text-primary-900 after:scale-x-100` -- use `usePathname()` from `next/navigation`. This requires `"use client"` directive.
  - Gap between links: `gap-8`
- Right (mobile): Hamburger button placeholder visible on `lg:hidden`. Just a Menu icon from lucide-react, size 24. `aria-label="Open menu"`. This button won't wire to anything yet (Plan 03 adds MobileNav).
- Import NAV_ITEMS from `@/lib/constants`
- Since usePathname is needed, add `"use client"` at top

**Footer (src/components/layout/Footer.tsx):**
- `<footer>` element with `bg-primary-950 text-stone-100`
- Max-w-7xl centered, padding `py-16 px-6`
- Layout: 4-column grid on desktop (`grid-cols-4`), stacking on mobile (`grid-cols-1 sm:grid-cols-2 lg:grid-cols-4`), gap-12
- Column 1 -- About: "St Katharine Rural Connect" heading (font-heading, text-xl, text-stone-50), short tagline from SITE_CONFIG, charity registration number in `text-stone-400 text-sm`
- Column 2 -- Quick Links: heading "Quick Links" in `text-accent-500 font-body font-semibold text-sm uppercase tracking-wider mb-4`. List first 5 NAV_ITEMS as links, `text-stone-300 hover:text-accent-400 transition-colors`
- Column 3 -- More Links: heading "More". List remaining 5 NAV_ITEMS. Same styling.
- Column 4 -- Contact: heading "Get in Touch". Phone number as tel: link, email as mailto: link, office hours, location. All from SITE_CONFIG.
- Below the grid: horizontal divider `border-t border-primary-800 mt-12 pt-8`
- Bottom row: flex between. Left: copyright "2026 St Katharine Rural Connect. Scottish Charity No. {charityNumber}". Right: social links from SOCIAL_LINKS -- render as lucide icons (Facebook, Twitter, Instagram) wrapped in `<a>` tags, `text-stone-400 hover:text-accent-400 transition-colors`, size 20, gap-4.
- Import NAV_ITEMS, SITE_CONFIG, SOCIAL_LINKS from `@/lib/constants`
- Import Facebook, Twitter, Instagram from lucide-react

Important: Use `font-heading` for headings and `font-body` for body text throughout. These are the Tailwind font family utilities configured in Phase 1 design tokens.
