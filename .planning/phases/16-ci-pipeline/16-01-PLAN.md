# Plan: 16-01 CI Pipeline

## Objective

Set up GitHub Actions CI workflow, ESLint flat config, comprehensive .gitignore, and Vercel auto-checks.

## Tasks

### Task 1: ESLint flat config
Create `eslint.config.mjs` using `eslint-config-next` flat config format for ESLint 9. Update `package.json` lint script to target `src/` directory. Verify lint passes.

**Files:** `eslint.config.mjs`, `package.json`

### Task 2: Comprehensive .gitignore
Replace minimal .gitignore with comprehensive coverage: .next/, node_modules/, env files, OS files (.DS_Store), editor configs, build artifacts, Vercel output.

**Files:** `.gitignore`

### Task 3: Add typecheck script
Add `"typecheck": "tsc --noEmit"` script to package.json for CI to call.

**Files:** `package.json`

### Task 4: GitHub Actions CI workflow
Create `.github/workflows/ci.yml`:
- Trigger: push to main, pull_request to main
- Node 22 (LTS), npm ci, cache node_modules
- Jobs: lint, typecheck, build (parallel)
- Use actions/checkout@v4, actions/setup-node@v4

**Files:** `.github/workflows/ci.yml`

### Task 5: Verify all CI steps pass locally
Run lint, typecheck, and build locally to confirm they all pass before committing.

## Success Criteria
1. `npm run lint` passes with zero errors
2. `npm run typecheck` passes
3. `npm run build` passes
4. `.github/workflows/ci.yml` exists with lint/typecheck/build jobs
5. `.gitignore` covers standard Next.js exclusions
